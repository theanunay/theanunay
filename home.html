1. Clipboard Group: -

(a). Cut: -
        When you want to remove the object from the document and use this object at another place then you can use this tool. When you use this tool then your selected object will be removed from there and store in the “Clipboard”.  

(b). Copy: -
       When you use this tool then your selected object will be copied in your clipboard. Then you can use this copied object as new object in your document, where you want to.  

(c). Paste: -
       To paste the cutted/copied object as new object in your document then you’ve to use this tool to paste the object.  

Objects can be anything like: - text, images, charts, shapes, SmartArt etc.

(d). Format Painter: -
       When you want to copy only applied formats of the object then you’ve to use this tool to do that.  

To Copy the Formats: -
       Select the object and select “Format Painter” from Clipboard group. (Then you’ll see your cursor like "Brush" in your document)  

To Paste the Formats: -
       Select that object which you want to apply this format using your mouse cursor and leave that to apply the format.  

2. Font Group: -

a). Bold: -
       To make your text to a heavier font. Its symbol is “B” and shortcut is “Ctrl+B”.  

b). Italic: -
       To italicize your text. Its symbol is "I" and shortcut is “Ctrl+I”.  

c). Underline: -
       To draw an underline below the text in your document. Its symbol is "U"" and shortcut is “Ctrl+U”.  

d). Strikethrough: -
       Cross something out by drawing a line through it. Its symbol is “abc”.  

e). Subscript: -
       Type very small letters just below the line of text without wrapping the line. It specially use in “Chemistry” to write an equation. Its symbol is “X2” and shortcut is “Ctrl+=”.  

f). Superscript: -
       Type very small letters just above the line of text without wrapping the line. It specially use in “Maths” to write a formula. Its symbol is "X2 and shortcut is “Ctrl+Shift+=”.  

g). Increase Font Size: -
       It is used to grow the font size of the selected text. Its shortcut key is “Ctrl+Shift+>”.  

h).Decrease Font Size: -
       It is used to decrease the font size of the selected text. Its shortcut key is “Ctrl+Shift+<”.  

i). Change Case: -
       Change the selected text to uppercase, lowercase, sentence case and all other common capitalizations. Its symbol is “Aa”.  

j). Text Effect & Typography: -
       It is used to format the text with special effects without inserting Word Arts & Text Boxes. You can do apply Glow, Shadow, Reflection, Outline effects to the text by selecting the text.  

k). Text Highlight Color: -
       Make your text a pop by highlighting it in a bright color. Its symbol is “ab” with a Brush. To do this,first of all select your text by keyboard or mouse cursor then click on Text Highlight Color option from Font group of Home Tab and choose the color that you want to apply as highlight color of the selected text.  

l). Font Color: -
       To change the color of the selected text with a desired color. To do this, select the text/word/paragraph then click on Font Color option and choose your color which you want to apply. Then you can see the changes in your document.  

m). Clear All Formatting: -
       Remove all formatting from the selection, leave only normal and unformatted text in the document. To do this, select the paragraph and then click on Clear All Formatting tool. Then you can see the changes in the document.  

3). Paragraph Group: -

       This group is used to set the margins and format the lists/multilevel lists by numbering or bullets. To setting up the line spacing and indent of the paragraph.  

a). Bullets: -
       To write one linear questions & answers or to write topics name without numbering then you’ve to use Bullets tool to apply this style.

To do this, you need to select Bullets option from Paragraph group of Home Tab and type the text and press “Enter” to create new line in the document. You’ll find next bullet in the newly created line automatically.

To deactivate this, you need to press “Enter” two time without typing the text.


b). Number List: -
       Number list is similar to bullets list. You can use this tool by same method. But, in this list you can use only number list instead of bullets. To do this, you need to select Number option from Paragraph Group of Home Tab and type the text and press “Enter” to create new line in the document. You’ll find next Number in the newly created line automatically.

To deactivate this, you need to press “Enter” two time without typing the text.


c). Multilevel List: -
       When you need to create list with level then you’ve to use this tool.

To do this, select “Multilevel list” from Paragraph Group from Home Tab and type anything in the first line then press “Enter” to split the row and then press “Tab Key” from Keyboard to create a level. If you want to create more level or go to next level you need to go to the new line and press “Tab Key” to change the level. To go back in the previous level you need to press “Enter” key double time without typing the text.


To Split the Row: - “Enter” one time
To Create a Level: - Press “Tab” Key one time.
To go back in the Previous Level: - Press “Enter” Key double time without typing the text.
d). Left Alignment: -
       To write from the left side of the page you need to set “Left Margin” from Paragraph group. Its shortcut key is “Ctrl+L”


e). Center Alignment: -
       To write in the center of the page, you need to set margin as “Center” from Paragraph group. Its shortcut key is “Ctrl+E”.


f). Righ Alignment: -
       To write from the right side of the page, you need to set margin as “Right”. Its shortcut key is “Ctrl+R”.


g). Justify Alignment: -
       To adjust the left & Right margin same then you need to set the page margin as “Justify”. When you set the Justify margin, your written document will be adjusted in proper letter format. Its shortcut key is “Ctrl+J”.

h). Increase Indent: -
       To set the paragraph margin you need to increase indent to increase space before (left side) paragraph. Its shortcut key is “Ctrl+M”.

i). Decrease Indent: -
       To decrease space before paragraph you need to use this tool. Its shortcut key is “Ctrl+Shift+M”.


j). Line Spacing: -
       To view the Rows & Paragraphs in separated, you need to set line spacing & Paragraph spacing from Paragraph Group. (There is no any shortcut to do it.) You can use “Add Space After Paragraph” option from “Line Spacing” tool to separate paragraphs.


k). Sort By: -
       Arrange the current selection in alphabetical or numerical order. This is especially useful if you’re trying to organize data in a table. To do this, select the data range then “Sort” tool then select “Ascending” or “Descending” to set the order.


l). Shading: -
       Change the color behind the selected text, paragraph or table cell. To do this, select the paragraph and select “Shading” tool and choose “color”. Then your selected color will be applied on the selected paragraph.


m). Border: -
       To separate the paragraph with a line, you’ve to apply “Border” in the selected paragraph. To remove the border, do the same.

To select the paragraph: - Ctrl+Shift+Down Navigation Key (If your cursor is before the paragraph).
To select the paragraph: - Ctrl+Shift+Up Nav Key ( If your cursor is after the paragraph).
To select the one word: - Ctrl+Shift+Forward Nav Key (When your cursor is before the word).
To select the word: - Ctrl+Shift+Backward Nav Key (When your cursor is after the word)

4). Styles Group: -

       Using it, you can apply a special effect in the document.

5). Editing Group: -

a). Find: -
       To find any specific word across the documetn you've to use this tool.

b). Replace: -
        It is used to replace any word or text in the document.

c). Select: -
        Using it, you can select any object in the document.
