An Introduction to Notepad App

        Notepad is an application which is used to create simple text document. It cannot perform any special Editing. This is a by default program of any computer with Windows Operating System.  
        Its file extension is *.txt . & its file save with name "Untitled-Notepad".  


        There is five menu available in Notepad App:- 1). File, 2). Edit, 3). Format, 4). View, & 5). Help  

1). File Menu: -
        It is used to get an output from the application after processing of the document. In which, you can print or save a file. You can also setup the page by "Page Setup option" available in File Menu.  

2). Edit Menu: -
        This menu is used to make various changes in the document. Like: - Undo,Replace, Inserting Time & Date etc. 

Uses of options available in Edit Menu: -

        a). Undo: - This function perform reverse action of an earlier action. For an example: - Undo option can turn back a deleted item.  

        b). Cut: -This function is used to cut the selected text and use as a new item in the document. You need to select first that text you want to cut then use Cut (Ctrl+X) then place your cursor where you want to Paste (Ctrl+V)it as new item. Once you used "Cut" command your selected text will be removed from the document.    

        c). Copy: -This function is used to copy the selected text and use as a new item in the document. You need to select first that text you want to Copy then use Copy (Ctrl+C) then place your cursor where you want to Paste (Ctrl+V)it as new item. When you use this function, there will be no any changes shown on the selected item.

        d). Paste: - When you use Cut or Copy command in the document then you need to paste the copied item, that time you have to use this function. Its shortcut key is Ctrl+V

        e). Find: - It is used to find the text across the document. To do this: - Select Find (Ctrl+F) option from the Edit Menu and type the text that you want to find and click Find Next option. You'll see your desired text with highlighted area.

        f). Replace: - It is used to replace the text with another text. To do this: - Select Replace option (Ctrl+H) from Edit Menu and type the text in the Find What section that you want to change. Then in the Replace With section, you have to type that text with you want to replace it.

        g). Select All: - Using it, you can select all document in one click. Its shortcut key is Ctrl+A.

        h). Date & Time: - Using this function you can insert Time & Date in the document. Its shortcut key is F5.

3). Format: -
        Using this option you can format the document. Like: - Wrap the text, Change the font size, Font Style, Font Written Style etc.

        a). Wrap Text: - When you enable this function your paragraph line will be automatically wrapped in the next line by splitting the line.

        b). Font: - In Notepad you can change your writing style using Font option. First of all Create a document using notepad then select Font option from Format Menu and apply the changes which you want to. Like: - Bold, Italic, Underline etc.

4). View: -
        Using it, you can do Zoom In & Zoom Out the page. You can also enable/disable Status Bar using it.

5). Help: -
        In all the applications, there is a menu called "Help", that help us to find support related to the application.
